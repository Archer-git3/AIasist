## 1. Встановлення Ollama
 
Ollama — це програма яка дозволяє запускати великі мовні моделі (LLM) локально на комп'ютері або сервері.
 
### Linux
```bash
curl -fsSL https://ollama.com/install.sh | sh
```

 
### Windows
Завантаж `.exe` інсталятор з [ollama.com/download](https://ollama.com/download) та встановлюється як звичайна програма.
 
---
 
## 2. Завантажити модель qwen3:14b
 
Після встановлення Ollama — завантаж модель на цей проєкт:
 
```bash
ollama pull qwen3:14b
```
 
> Модель важить ~9 GB.
 
Перевірити що модель завантажена:
```bash
ollama list
# Має показати: qwen3:14b
```
 
### Системні вимоги для qwen3:14b
 
| Компонент | Мінімум | Рекомендовано |
|-----------|---------|---------------|
| RAM | 16 GB | 32 GB |
| Диск | 12 GB вільного місця | 20 GB |
| GPU | не обов'язково | NVIDIA 8GB+ / Apple Silicon |
 
> Якщо RAM не вистачає — можна використати легшу модель:
> ```bash
> ollama pull qwen3:8b   # ~5 GB, менш точна
> ollama pull qwen3:4b   # ~2.5 GB, для слабких машин
>```
 
## 3. Запустити Ollama (якщо не запускається автоматично)
 
```bash
ollama serve
```
 
Перевірити що сервер працює:
```bash
curl http://localhost:11434/api/tags
# Має повернути список моделей у JSON
```
